# 🌱 Сайт о Саженцах

Приветствую тебя мой друг, ты можешь посмотреть мой сайт, посвящённый теме саженцев.

## 📦 О проекте

Этот проект был создан с использованием чистых технологий:

- **HTML**: Структура и содержание страниц.
- **CSS**: Стилизация и оформление.
- **JavaScript**: Добавление интерактивности и динамического контента.

## 🚀 Деплой

Сайт развернут на платформе [Vercel](https://sad-rose.vercel.app/).
